# Web Dev Helper Skill

When building websites:

1. Use clean, beginner-friendly code.
2. Create separate files for HTML, CSS, and JavaScript unless asked otherwise.
3. Make the design modern, responsive, and mobile-friendly.
4. Explain the purpose of each file.
5. Avoid overcomplicating the code.
6. When fixing errors, explain the mistake first, then fix it.
7. Include comments only where they help learning.
8. For every website, try to include:
   - navigation bar
   - hero section
   - main content
   - footer
   - responsive CSS
